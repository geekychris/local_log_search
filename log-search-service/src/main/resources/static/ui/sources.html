Grok Pattern Quick Reference

Common Patterns

Apache/Nginx Access Logs 
{"pattern": "%{COMBINEDAPACHELOG}"}
Example: 127.0.0.1 - frank [10/Oct/2000:13:55:36 -0700] "GET /page.html HTTP/1.0" 200 2326
Syslog Format 
{"pattern": "%{SYSLOGBASE} %{GREEDYDATA:message}"}
Example: Mar 16 00:01:25 hostname sshd[1234]: Failed password for user
Timestamp + Level + Message 
{"pattern": "%{TIMESTAMP_ISO8601:timestamp} %{LOGLEVEL:level} %{GREEDYDATA:message}"}
Example: 2024-01-15T10:30:00Z INFO Application started successfully
Custom Key=Value Format 
{"pattern": "%{TIMESTAMP_ISO8601:timestamp} user=%{USERNAME:user} ip=%{IP:ip} action=%{WORD:action}"}
Example: 2024-01-15T10:30:00Z user=john.doe ip=192.168.1.50 action=login

Common Pattern Components

Pattern Matches
%{TIMESTAMP_ISO8601}ISO timestamps
%{LOGLEVEL}DEBUG, INFO, WARN, ERROR
%{IP}IP addresses
%{USERNAME}Usernames
%{NUMBER}Any number
%{WORD}Single word
%{GREEDYDATA}Everything to end of line
